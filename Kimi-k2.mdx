## Quick Setup

<Steps>
  <Step title="Get API Access">
    Choose your preferred provider:
    - **OpenRouter** (recommended): Visit [openrouter.ai](https://openrouter.ai)
    - **Direct Moonshot**: Visit [platform.moonshot.ai](https://platform.moonshot.ai)
  </Step>
  
  <Step title="Install Claude Code Router">
    ```bash
    npm install -g @musistudio/claude-code-router
    ```
  </Step>
  
  <Step title="Configure Router">
    ```bash
    ccr start
    ```
    Enter your configuration:
    ```
    Provider: openrouter
    API URL: https://openrouter.ai/api/v1
    API Key: sk-or-v1-your-key
    Model: moonshotai/kimi-k2
    ```
  </Step>
  
  <Step title="Test Integration">
    ```bash
    ccr code "Create a Python web scraper with error handling"
    ```
  </Step>
</Steps>

## Key Capabilities

### Code Generation
Generate complete, working applications in a single request:

```bash
ccr code "Create a complete todo app with React, TypeScript, and local storage"
```

### Autonomous Debugging
Kimi K2 can identify, analyze, and fix bugs automatically:

```bash
ccr code "Debug and fix all TypeScript errors in this codebase"
```

### Workflow Automation
Automate complex development workflows:

```bash
ccr code "Set up CI/CD pipeline with testing, linting, and deployment"
```

## Integration Options

<Tabs>
  <Tab title="Qwen CLI">
    Use Kimi K2 through Qwen CLI for enhanced code-aware conversations:
    ```bash
    npm install -g @qwen-code/qwen-code
    qwen # Configure with OpenRouter + Kimi K2
    ```
  </Tab>
  
  <Tab title="Direct API">
    Integrate directly in your applications:
    ```javascript
    const response = await fetch('https://openrouter.ai/api/v1/chat/completions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        model: 'moonshotai/kimi-k2',
        messages: [{ role: 'user', content: 'Your prompt' }]
      })
    });
    ```
  </Tab>
  
  <Tab title="IDE Integration">
    Use with AI-powered IDEs like Windsurf for seamless development experience.
  </Tab>
</Tabs>
