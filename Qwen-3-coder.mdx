# Qwen 3 Coder Integration

Qwen 3 Coder is a powerful code-focused language model designed for software development tasks. This guide covers how to integrate it with Qwen CLI and other development tools.

## Overview

Qwen 3 Coder excels at:
- Code generation and completion
- Bug fixing and debugging
- Code refactoring and optimization
- Technical documentation
- Code review and analysis

## Quick Setup

<Steps>
  <Step title="Install Qwen CLI">
    ```bash
    npm install -g @qwen-code/qwen-code
    ```
  </Step>
  
  <Step title="Configure Authentication">
    Run `qwen` and set up your API credentials:
    ```
    API Key: your-qwen-api-key
    Base URL: https://api.qwen.com/v1
    Model: qwen3-coder
    ```
  </Step>
  
  <Step title="Test Integration">
    ```bash
    qwen "Write a Python function to sort a list of dictionaries by a specific key"
    ```
  </Step>
</Steps>

